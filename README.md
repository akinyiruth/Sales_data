📊 Mini Sales Performance Dashboard
📌 Project Overview

This project showcases an interactive Excel dashboard used to analyze sales performance by product and region. It demonstrates my skills in data cleaning, analysis, and visualization using Microsoft Excel.

📁 Dataset
Small dataset (less than 15 records)
Fields included: Product, Date, Product category,Region, Sales, Sales Representative

⚙️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers

📈 Key Insights
Nairobi has the highest sales
Shampoo is the best-performing product
Sales vary across regions
